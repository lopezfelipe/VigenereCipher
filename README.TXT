PROJECT TITLE
Breaking the Vigenere Cipher

PURPOSE OF PROJECT:
Decrypt messages from unkown languages with keys of unknown length.

VERSION or DATE:
January 2018

HOW TO START THIS PROJECT:
(1) Create a new VigenereBreaker object
(2) Load the dictionaries of the languages to evaluate
(3) Run breakVigenere on the VigenereBreaker object
(4) Select the encrypted file
